# JOO

A React Native finance app built with Expo.

## Stack

- [Expo](https://expo.dev) ~54
- React Native 0.81
- TypeScript (strict)
- New Architecture enabled

## Getting Started

Install dependencies:

```bash
npm install
```

Run on your device (requires [Expo Go](https://expo.dev/go)):

```bash
npx expo start
```

Scan the QR code with your camera (iOS) or the Expo Go app (Android).

| Command | Description |
|---|---|
| `npx expo start --ios` | Open in iOS Simulator |
| `npx expo start --android` | Open in Android emulator |
| `npx expo start --web` | Open in browser |
